# Nombre del proyecto: Triple Espresso - coffe shop

# Autor: Raymundo Flores

# Descripción del proyecto: Aquí tenemos una pagina dedicada a una cafetería, la cual contiene cuatro secciones, las cuales son las siguientes:

1.- El mensaje de bienvenida a la pagina, con las secciones disponibles en la parte superior derecha, horarios de atención en la esquina inferior izquierda y la dirección donde se encuentra ubicada la cafetería colocada en la parte inferior derecha.

2.- Recetas: en esta sección se agregaron dos videos que muestran la preparación de dos tipos de cafés.

3.- Reserva una mes: la cual contiene un pequeño formulario, el cual ayudara a reservar de mejor manera una mesa dentro de la cafetería.

4.- Contáctanos: esta es la ultima sección de la pagina creada, agregando en ella las redes sociales de la cafetería.

# Tecnologías utilizadas:

- HTML5
- CSS3
- Metodología BEM
- flexbox
- Grid Layout
